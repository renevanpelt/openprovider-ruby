```shell

```
(for development, run `gem install --dev ./openprovider-ruby-1.0.0.gem` to install the development dependencies)

or publish the gem to a gem hosting service, e.g. [RubyGems](https://rubygems.org/).

Finally add this to the Gemfile:

    gem 'openprovider-ruby', '~> 1.0.0'
